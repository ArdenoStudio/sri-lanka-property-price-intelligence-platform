serp_property

Primary public search. Paginated property ads (~25/page).

search GET /v1/serp

Last probe: ✅ 200

curl

curl -sS 'https://api.ikman.lk/v1/serp?category=409&page=1' \
  -H 'Accept: application/json' \
  -H 'Application: web'

Notable fields

  • pagination.page
  • pagination.per_page
  • pagination.total
  • pagination.pages
  • pagination.next_page_token
  • serp.types[]
  • serp.categories[]
  • serp.locations[]
  • serp.results[].id — 24-char hex ad id
  • serp.results[].title
  • serp.results[].money.amount
  • serp.results[].details[]
  • serp.results[].images

Notes

  • page ≥ 2 usually needs next_page_token from prior response
  • Island-wide deep pages (~450–500) often HTTP 500 — location-shard
  • Active inventory only; not a multi-year archive
