locations

District/city tree with slug and geo_region (e.g. LK-11).

meta GET /v1/locations

Last probe: ✅ 200

curl

curl -sS 'https://api.ikman.lk/v1/locations' \
  -H 'Accept: application/json' \
  -H 'Application: web'

Notable fields

  • locations[].id
  • locations[].name
  • locations[].slug
  • locations[].geo_region
  • locations[].children[]
