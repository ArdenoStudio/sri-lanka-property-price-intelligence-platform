ad_detail

Single ad detail — description, properties[], statistics.views, similar_ads.

detail GET /v1/ads/{{ad_id}}

Last probe: ✅ 200

curl

curl -sS 'https://api.ikman.lk/v1/ads/<ad_id>' \
  -H 'Accept: application/json' \
  -H 'Application: web'

Notable fields

  • ad.id
  • ad.title
  • ad.description
  • ad.properties[] — key/label/value for bedrooms, bathrooms, size, etc.
  • ad.statistics.views
  • similar_ads[]
  • safety_tips[]
